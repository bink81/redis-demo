# redis-demo
This is a simple demo of a REST API that uses redis

## Installation

### To start local server:
```
mvn spring-boot:run
```

### To package:
```
mvn package
```

## Operations

### View/add messages

When started, the system has a REST API that supports GET and POST requests:

http://localhost:8080/messages/
